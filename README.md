# DynamicNetworkObserver

### A plug-and-play Android library to detect network changes and show offline UI.

## Features
- Offline Banner
- Offline Full Screen Mode
- Retry logic
- Flow + Coroutines based
- Works in XML & Compose
- 0 dependencies
